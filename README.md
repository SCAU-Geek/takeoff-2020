## geek2020-takeoff

#### 本项目所用技术栈：

1. spring boot
2. okhttp3

#### 打包方法：

1. git clone https://github.com/SCAU-Geek/takeoff-2020.git
2. mvn clean package

#### 运行方法：

打包之后

1. cd target/
2. java -jar geek2020-0.0.1-SNAPSHOT.jar

#### 使用方法:

1. curl http://127.0.0.1/showname?number=your_student_number&password=your_password
2. 使用http请求工具
